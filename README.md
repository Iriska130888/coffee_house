# Coffee House App

Проект представляет собой веб-приложение для кофейни с фронтенд-частью на Vite и бэкендом на Django.

## Основной функционал

- Динамическое меню с категориями.
- Система оформления заказов.
- Модуль бронирования столиков.
- Панель администратора для управления контентом.

## Технические характеристики

- Бэкенд: Django, Django Rest Framework.
- База данных: SQLite.
- CI/CD: GitHub Actions (автоматическое тестирование кода при пуше).

## Спецификация API

- GET `/api/menu/` — получение списка блюд.
- GET `/api/categories/` — список категорий.
- POST `/api/reservations/` — создание бронирования.
- POST `/api/orders/` — оформление заказа.

## Инструкция по запуску

1. Установить зависимости: pip install -r requirements.txt
2. Выполнить миграции: python manage.py migrate
3. Запустить локальный сервер: python manage.py runserver
